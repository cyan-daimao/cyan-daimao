关于我

数据平台架构

专注于湖仓一体架构设计与实现，从数据采集、存储治理到智能分析的全链路数据平台建设。

PROJECTS

精选项目

rustfs + iceberg + restcatalog

湖仓一体存储引擎，存算分离架构。RustFS 自研对象存储 + Iceberg 开放表格式 + REST Catalog 统一元数据接口。

Rust Iceberg REST Catalog 对象存储

cyan-dataman

数据采集 + 元数据平台。CDC 数据入湖链路（Debezium + Kafka + Flink），以及基于 Gravitino + Iceberg 的元数据治理。

Java Flink Kafka Gravitino

cyan-datagateway

数据网关，StarRocks / Spark / Flink 的统一 SQL 执行入口。智能路由、资源分配、SQL 解析增强。

Java StarRocks Spark SQL 路由

cyan-dataauth

数据权限中心，提供表级、行级、列级权限控制。对外权限申请，对内 SQL 增强服务。

Java Spring Boot 权限管控 SQL 改写

cyan-dataworks

调度系统（规划中）。Spark / Flink 作业编排调度，依赖表 + 版本号记录血缘，Agent 智能运维。

Java Spark Flink 数据血缘

cyan-datametric

指标平台，数据应用服务的基石。指标维度管理、DSL 查询语言，通过关联关系图谱实现智能聚合。

Java DSL 指标聚合 图谱计算

cyan-databi

智能分析平台，基于指标平台创建图表，支持 ChatBI 自然语言生成图表。图表 + 布局组成数据看板。

React TypeScript ChatBI 数据看板
